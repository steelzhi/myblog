Приложение-блог с использованием Spring Framework и тестами

Инструкция:
1. В IDE собрать проект командной "mvn clean install".
2. Скачать контейнер сервлетов "Tomcat". Запустить его: папка "bin" -> startup.
3. Собранный в п. 1 проект myblog.war скопировать в папку "Tomcat" "webapps". Через несколько секунд контейнер будет
распакован и готов к работе.
4. В папке "Tomcat" "conf" в файле "tomcat-users" раскомментировать пользователя с username = "admin".
5. Перейти по ссылке: localhost:8080/manager/html. Ввести логин "admin" и пароль "admin".
6. Перейти по ссылке: localhost:8080/myblog/feed. Теперь Вы находитесь на странице ленты и можете совершать действия,
описанные ниже.

В приложении реализовано:
1. Добавление нового поста (с названием, картинкой, текстом, тегами);
2. Пагинация постов в ленте (помимо пагинации на 10, 20 и 50 постов, добавлена пагинация на 2 поста - 
для удобства тестирования);
3. Редактирование и удаление поста;
4. Возможность ставить лайки посту;
5. Возможность добавлять комментарии посту. А также - редактировать и удалять оставленные комментарии.
6. Поиск по тегу (символ "#" при этом ставить необязательно).

Особенности:
- При добавлении нового поста теги необходимо перечислить через запятую (знак # у тега при этом необязателен -
при его отсутствии программа сама его добавит);
- Для целей более полного тестирования в репозитории методы добавления, изменения и удаления тоже возвращают объекты;
- Комментарии редактируются непосредственно в поле комментария. Для сохранения отредактированного комментария нужно нажать комбинацию Ctrl + Enter

Контроллеры, сервисы, репозитории, маппер покрыты интеграционными и юнит-тестами.
