Приложение-блог с использованием Spring Framework и тестами

Инструкция:


В приложении реализовано:
1. Добавление нового поста (с названием, картинкой, текстом, тегами);
2. Пагинация постов в ленте (помимо пагинации на 10, 20 и 50 постов добавлена пагинация на 2 поста - 
3. для удобства тестирования);
4. Редакторание и удаление поста;
5. Возможность ставить лайки посту;
6. Возможность добавлять комментарии посту. А также - редактировать и удалять оставленные комментарии.

Контроллеры, сервисы, маппер покрыты интеграционными и юнит-тестами.
