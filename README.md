Приложение-блог с использованием Spring Boot и тестами

Инструкция:
1. Запуск программы:
    * 1.1. Вариант 1: запустить класс MyblogBootApplication.
    * 1.2. Вариант 2: собрать проект в jar-файл одной из команд:
        * 1.2.1. `gradle clean build`;
        * 1.2.2. `./gradlew clean build`
    Перейти в "build" (оранжевая папка) -> "libs" и скопировать полный путь файла "myblog-boot-0.0.1-SNAPSHOT.jar". В
    консоли ввести команду: java -jar {полный путь файла "myblog-boot-0.0.1-SNAPSHOT.jar"}
    Примечание: если в названии папок пути есть проблемы, весь путь нужно взять в двойные кавычки.
2. Перейти по ссылке: localhost:8080/feed. Теперь Вы находитесь на странице ленты и можете совершать действия,
описанные ниже.

В приложении реализовано:
1. Добавление нового поста (с названием, картинкой, текстом, тегами);
2. Пагинация постов в ленте (помимо пагинации на 10, 20 и 50 постов, добавлена пагинация на 2 поста - 
для удобства тестирования);
3. Редактирование и удаление поста;
4. Возможность ставить лайки посту;
5. Возможность добавлять комментарии посту. А также - редактировать и удалять оставленные комментарии.
6. Поиск по тегу (символ "#" при этом ставить необязательно).

Особенности:
- По умолчанию задано количество постов на странице;
- При добавлении нового поста теги необходимо перечислить через запятую (знак # у тега при этом необязателен -
при его отсутствии программа сама его добавит);
- Для целей более полного тестирования в репозитории методы добавления, изменения и удаления тоже возвращают объекты;
- Комментарии редактируются непосредственно в поле комментария. Для сохранения отредактированного комментария нужно 
нажать комбинацию Ctrl + Enter.

Контроллеры, сервисы, репозитории, маппер покрыты интеграционными и юнит-тестами. Контроллерам написаны тесты как 
с использованием Mock-объектов (тестируется только непосредственно слой контроллеров), так и без них (тестируются 
сразу все слои).
