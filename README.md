Приложение-блог с использованием Spring Framework и тестами

Инструкция:


В приложении реализовано:
1. Добавление нового поста (с названием, картинкой, текстом, тегами);
2. Пагинация постов в ленте (помимо пагинации на 10, 20 и 50 постов, добавлена пагинация на 2 поста - 
для удобства тестирования);
3. Редакторание и удаление поста;
4. Возможность ставить лайки посту;
5. Возможность добавлять комментарии посту. А также - редактировать и удалять оставленные комментарии.

Особенности:
- При добавлении нового поста теги необходимо перечислить через запятую (знак # у тега при этом необязателен -
при его отсутствии программа сама его добавит);
- Для целей более полного тестирования в репозитории методы добавления, изменения и удаления тоже возвращают объекты

Контроллеры, сервисы, репозитории, маппер покрыты интеграционными и юнит-тестами.
